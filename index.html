Changelog v0.2 (English)

System & Kernel

  • GZ Mode integrated: The Ghost Zone (gz) command was moved to /usr/local/bin and execution permissions (755) are enforced via profiledef.sh.
  • Build optimizations: Automatic cleanup routines were added to handle work folder and /proc mount issues during builds.

Interface & Terminal

  • Terminal identity: Replaced the Kali-inspired theme with a minimalist red-white [RedRose] $ prompt running on zsh.
  • Font update: Fixed rendering issues and switched to Noto Sans and JetBrains Mono Nerd Font for bolder, clearer typography.

Visual Design

  • New splash screen: Boot logo updated to a GRUB-compatible 8-bit (256-color) format.
  • Custom icon set: System-specific redrose_icon.png icons and application artwork added.
  • Default wallpaper: A black-red themed default wallpaper is included for Snapshot 0.2.

About RedRose OS

RedRose is a security-focused live distribution built on an Arch Linux rolling base. This snapshot (v0.1) is intended for research, education and short-lived investigative sessions.

Features

  1. Arch-Linux Performance
    • Rolling-release base: Built on the latest Linux kernel to stay current, fast and stable.
    • XFCE professional interface: Lightweight, performance-oriented desktop with a burgundy aesthetic tailored for security work.
  2. Seamless Live Experience
    • Zero-trace operation: Runs entirely in RAM — remove the USB and no files, logs or traces remain on the host.
    • No-password access: Live boot gives immediate root access for fast testing and incident response workflows.
  3. Integrated Tools & Framework
    • Unified security workspace: Tools and configurations are optimised to work together for a plug-and-play lab experience.
    • High-speed boot architecture: Custom boot optimisations to reduce time-to-operation.
  4. Hardware Compatibility
    • x86_64 precision: Optimised for modern 64-bit processors to maximise performance.
Note:

This is a snapshot release — it includes basic tools such as nmap and does not ship package repositories. Future releases will include an installable ISO alongside the live image. Mobile NetHunter-style capabilities may be explored in the future.
